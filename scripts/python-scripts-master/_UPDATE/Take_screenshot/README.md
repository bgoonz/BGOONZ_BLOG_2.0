# Take a Screenshot :

A simple implementation on how to take screenshots .

#### Required Modules :

- Numpy
  ```bash
    pip install numpy
  ```
- Opencv
  ```bash
    pip install opencv-python
  ```
- Pyautogui
  ```bash
    pip3 install pyautogui --user
  ```
