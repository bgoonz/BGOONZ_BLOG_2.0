## Share links

{: .-one-column}

Facebook:
{:.-setup}

```html
<a href="https://www.facebook.com/sharer/sharer.php?u=URL" target="share"></a>
```

Twitter:
{:.-setup}

```html
<a href="https://twitter.com/intent/tweet?text=DESCRIPTION+URL" target="share"></a>
```

Google Plus:
{:.-setup}

```html
<a href="https://plus.google.com/share?url=URL" target="share"></a>
```
