Here's a 4-line snippet on how to get beautiful, antialiased text with CSS.

---

### Antialias

```css
* {
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
}
```

### Support

-   Firefox 25+ on OSX
-   Webkits (Chrome, Safari, etc)
